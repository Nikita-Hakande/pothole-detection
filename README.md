# Pothole Detection System

This project implements a pothole detection system using deep learning.
The model is trained to identify potholes from road images to improve
road safety and infrastructure monitoring.

## Technologies Used
- Python
- CNN / Deep Learning
- OpenCV
- Google Colab

## How to Run
- Open the notebook in Google Colab
- Upload dataset
- Run all cells sequentially

## Project Type
Academic / Machine Learning Project
